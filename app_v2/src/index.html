DevOps Prject

This Project is what is currently hosting this website. It uses AWS to host the Elastic Kubernetes Service (EKS), and the EKS cluster deploys this site. To deploy the EKS cluster I used Terraform to construct the infrastructure as code where I applied the code to AWS to create the EKS cluster. I used docker to create an image and push it to AWS's Elastic Container Registry where my EKS cluster used the image to deploy this webapp.
